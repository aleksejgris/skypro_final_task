Git

  • git init - создание нового репозитория Git в пустом директории или превращение существующей директории в новый репозиторий Git.
  • git clone - клонирование репозитория из удаленого хранилища на локальную машину.
  • git push - выгрузка изменений в удаленый репозиторий.
  • git commit - сохранения изменения в Git.
  • git status - просмотр списка измененых файлов/
